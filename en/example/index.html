Instructions:

  • Please click the button above to open the video player and display the annotations.
  • Click an annotation to play the corresponding part of the video.
  • Hover over the annotations for details.
  • Some annotations are associated to certain keywords or references. You can highlight them by clicking the buttons below the video.
  • So far only annotated punches can be viewed. Although the data model supports types like: evasions, movement, stance, and more.
  • There's an event listener trying to mimic YouTube's keyboard shortcuts when there's no focus on the iframe (or else the YT API handles that itself). Not all shortcuts will work right now though.
  • I haven't tested this with every browser, but it should run in the main browsers.
  • Beware: The slider/progress bar tends to hiccup. If this happens, try to use pause/play or click an annotation. This should reset the slider.
